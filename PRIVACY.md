# COE Privacy (beta)

We collect minimal data to run the service:

- Device public keys (identity + sign) at enroll
- Daily key issuance logs (no raw keys)
- IP for rate limiting / abuse
- Optional crash reports (no secrets)

We do not:
- See message content (end-to-end)
- Sell data
- Share with third parties except legal requirement

Data retention: keys until revoked or device inactive; logs 90 days.

Contact: see repo for support.
